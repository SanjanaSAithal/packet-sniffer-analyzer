# Packet Sniffer + Analyzer (Wireshark-lite)

A packet sniffer built in C on Kali Linux (UTM) to capture and analyze network packets.
This project is for strengthening core CS concepts:
- Computer Networks
- Operating Systems
- Cybersecurity basics

## Planned Features
- Capture raw packets
- Parse Ethernet + IPv4 headers
- Decode TCP / UDP headers
- Filter by protocol / port
- Save logs to a file
- Basic anomaly detection (optional)

## Tech Stack
- C
- Kali Linux (UTM on macOS)
- GCC
- Wireshark (verification)
